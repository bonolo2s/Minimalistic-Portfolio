# Bonolo Lloyd Selowa — Developer Portfolio

A personal portfolio built to showcase my best work, skills, and experience as a Full-Stack Software Developer.

---

## Why I Built This

I wanted a space that truly represents the quality of work I deliver — not a template, but something built from scratch that reflects how I approach every project: with precision, reliability, and attention to detail.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| UI Components | shadcn/ui + Radix UI primitives |
| Forms | React Hook Form + Zod |
| Contact | Formsubmit.co |
| Charts | Recharts |
| Icons | Lucide React |
| Analytics | Vercel Analytics |
| Deployment | Netlify |

---

## Features

- Fully responsive across all screen sizes
- Smooth scroll-triggered animations via Intersection Observer
- Skills section with categorised tech chips
- Experience timeline with detailed bullet points
- Projects section with background image previews
- Working contact form (First Name, Last Name, Email, Message) via Formsubmit.co
- Dark/light theme support via next-themes

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
